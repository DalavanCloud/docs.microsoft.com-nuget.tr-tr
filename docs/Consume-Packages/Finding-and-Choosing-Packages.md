## <a name="search-syntax"></a><span data-ttu-id="a3447-179">Search sözdizimi</span><span class="sxs-lookup"><span data-stu-id="a3447-179">Search Syntax</span></span>

<span data-ttu-id="a3447-180">NuGet paket arama aynı nuget.org, NuGet clı'dan ve Visual Studio'da NuGet Paket Yöneticisi uzantısı içinde çalışır.</span><span class="sxs-lookup"><span data-stu-id="a3447-180">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="a3447-181">Genel olarak, arama paket açıklamaları yanı sıra anahtar sözcükleri uygulanır.</span><span class="sxs-lookup"><span data-stu-id="a3447-181">In general, search is applied to keywords as well as package descriptions.</span></span>

- <span data-ttu-id="a3447-182">**Anahtar sözcükler**: arama için sağlanan tüm anahtar sözcükler içeren ilgili paketleri arar.</span><span class="sxs-lookup"><span data-stu-id="a3447-182">**Keywords**: Search looks for relevant packages that contain all the provided keywords.</span></span> <span data-ttu-id="a3447-183">Örnek: `modern UI javascript`</span><span class="sxs-lookup"><span data-stu-id="a3447-183">Example: `modern UI javascript`</span></span>
- <span data-ttu-id="a3447-184">**Tümcecikleri**: tırnak işaretleri içindeki koşulları girmeniz arar bu koşulları büyük küçük harf duyarsız tam eşleşme.</span><span class="sxs-lookup"><span data-stu-id="a3447-184">**Phrases**: Entering terms within quotation marks looks for exact case-insensitive matches to those terms.</span></span> <span data-ttu-id="a3447-185">Örnek: `"modern UI" package`</span><span class="sxs-lookup"><span data-stu-id="a3447-185">Example: `"modern UI" package`</span></span>
- <span data-ttu-id="a3447-186">**Filtreleme**: sözdizimini kullanarak belirli bir özellik için bir arama terimi uygulayabilirsiniz `<property>:<term>` nerede `<property>` (büyük küçük harf duyarsız) olabilir `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, ve `owner`.</span><span class="sxs-lookup"><span data-stu-id="a3447-186">**Filtering**: You can apply a search term to a specific property by using the syntax `<property>:<term>` where `<property>` (case-insensitive) can be `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, and `owner`.</span></span> <span data-ttu-id="a3447-187">Koşulları gerekirse tırnak içine bulunabilir ve aynı anda birden çok özellikleri için arama yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="a3447-187">Terms can be contained in quotes if needed, and you can search for multiple properties at the same time.</span></span> <span data-ttu-id="a3447-188">Ayrıca, üzerinde arar `id` özelliği olan eşleşmelerini, ancak `packageid` tam bir eşleşme kullanır.</span><span class="sxs-lookup"><span data-stu-id="a3447-188">Also, searches on the `id` property are substring matches, whereas `packageid` uses an exact match.</span></span> <span data-ttu-id="a3447-189">Örnekler:</span><span class="sxs-lookup"><span data-stu-id="a3447-189">Examples:</span></span>

    ```
    id:NuGet.Core                # Match any part of the id property
    Id:"Nuget.Core"
    ID:jQuery
    title:jquery                 # Searches title as shown on the package listing
    PackageId:jquery             # Match the package id exactly
    id:jquery id:ui              # Search for multiple terms in the id
    id:jquery tags:validation    # Search multiple properties
    id:"jquery.ui"               # Phrase search
    invalid:jquery ui            # Unsupported properties are ignored, so this
                                 # is the same as searching on jquery ui
    ```
